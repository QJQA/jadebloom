# JADÉ BLOOM 水晶手串设计工坊

这是 `crystal-bracelet-designer` 的本地视觉与交互升级版。当前版本只保存在本机，尚未提交或部署到线上。

## 本机预览

在 macOS 中双击 `启动本地预览.command`，浏览器会自动打开：

`http://127.0.0.1:8765`

关闭启动器的终端窗口即可停止本地预览。

## 本次升级

- 现代东方珠宝工坊视觉系统
- 稳定的桌面双栏布局与移动端响应式布局
- 更自然的晶石高光、透光、纹理与阴影
- 自由设计、灵感配方与全库搜索
- 珠子可持续添加，画布随数量自动适配；实际尺寸在线下试戴确认
- 单颗编辑、拖动排序、撤销与重做
- 本机自动保存与双确认清空
- 4:5 品牌设计卡导出
- 键盘焦点、语义标签和减少动态效果支持
